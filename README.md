# Summertime Saga Mod Installation Guide

A practical guide to installing mods in Summertime Saga on both Android and PC, including correct file placement, version compatibility, and conflict prevention.

This guide helps you install mods safely without breaking your game or losing progress.

## Before Installing Any Mod

Mods modify core game files, so preparation is important. Skipping basic steps is the main reason players face crashes, black screens, or corrupted saves.

## Important Steps

### Back Up Your Save Files
Before installing any mod, create a backup of your save data.

- On PC: `%appdata%/RenPy/SummertimeSaga`
- Copy this folder to a safe location

### Match Mod Version to Game Version
Every mod is built for a specific version of the game.

- Using the wrong version can cause crashes  
- Always check compatibility before installing  

### Install One Mod at a Time
Avoid installing multiple mods together.

- Add one mod  
- Test the game  
- Then proceed to the next  

This makes it easier to detect conflicts.

### Download from Trusted Sources
Only use mods from verified fan communities.

- Avoid unknown APK files  
- Unverified downloads may contain harmful or broken files  

## Common Mod Issues

- Game crashes after install  
- Black screen on launch  
- Missing or broken scenes  
- Save file not loading  

Most of these issues are caused by version mismatch or incorrect file placement.
